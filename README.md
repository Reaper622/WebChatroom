# WebChatroom
使用HTML + CSS + Node.js + express 实现的在线web聊天室

> ### 实现功能：
- 登录
- 发送消息
- 查看源码
- 注销登录
- 查看时间

> ### 待实现功能:
- 发送表情
- 实现数据库存储用户信息
- 实现登录前验证用户
- 实现统计在线人数
- 实现**关于**功能
- 实现**打赏**功能
- 实现显示在线用户的功能

### 运行代码
```
git clone 仓库地址
```

进入根目录
```
node server.js
```
当提示server is running 时，访问[localhost:3000](http://localhost:3000/login.html)
